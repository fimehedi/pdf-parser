## Data flow

### Inputs
- `input.pdf`

### Intermediate artifacts (for reproducibility)
- `out/artifacts/page_XXXX.png`: rendered page image
- `out/artifacts/page_XXXX_pre.png`: preprocessed page image

### Canonical representation (CIR)
- `out/document.json` using schema `schemas/canonical_document_schema.json`

### Derived exports
- `out/document.md`
- `out/document.html`
- `out/tables/*.csv`
