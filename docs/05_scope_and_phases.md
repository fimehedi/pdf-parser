## Scope and phases

### Phase 1 (current)
- Scanned-first pipeline with Tesseract BN+EN
- Simple layout blocks + 1–2 column ordering
- Image-based table detection + CSV export
- JSON/MD/HTML exporters
- Confidence scoring v1

### Phase 2
- Better layout detection (headers/footers/sidebars) and block typing
- Figure detection + caption linking improvements
- Page-parallel processing + caching
- Optional OCR engines (EasyOCR/PaddleOCR) as configurable fallback

### Phase 3
- Semantic extraction for educational artifacts:
  - question/answer segmentation
  - learning objectives
  - quiz/homework templates
