## User stories

### Content operator
- As a content operator, I can parse a scanned textbook PDF and obtain structured text blocks and tables so I can build quizzes and homework faster.
- As a content operator, I can view an HTML output with confidence indicators so I know where to review.

### QA reviewer
- As a reviewer, I can locate low-confidence blocks/pages and compare with page images to correct mistakes.
- As a reviewer, I can export tables as CSV for reuse in worksheets.

### Platform engineer
- As an engineer, I can run the parser headlessly (CLI/Docker) and store outputs for downstream tools.
- As an engineer, I can rely on a stable canonical JSON schema to integrate with other services.
